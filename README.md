# Pixel Block Puzzle 🧩

블록을 끌어 줄을 맞추고, 그 블록이 위로 올라가 **픽셀 그림**을 지워 완성하는 캐주얼 퍼즐 게임.
외부 라이브러리 없이 **순수 HTML/CSS/JavaScript 단일 파일**(`pixel_block_puzzle.html`)로 동작합니다.

▶ **바로 플레이:** https://applepang-cloud.github.io/pixel-block-puzzle/
💻 **저장소:** https://github.com/applepang-cloud/pixel-block-puzzle

레벨을 깨며 **유리**와 **준**의 풋풋한 연애 이야기가 한 컷씩 펼쳐지고, 플레이로 모은 색과 코인으로
**픽셀 컬러링**과 **방 꾸미기**를 즐길 수 있어요. ♥

---

## 게임 방법

1. 화면 **아래 블록을 끌어다** 가로/세로 한 줄을 꽉 채우면 줄이 **펑** 터집니다.
2. 터진 색 블록이 **상단 보드 둘레의 벨트를 빙글빙글 돌며** 같은 색 픽셀을 쏴서 지웁니다.
3. 상단 그림을 모두 지우면 **레벨 클리어!**
4. 못 맞춘 블록은 **대기열**로 가고, 대기열이 가득 차거나 블록 놓을 곳이 없으면 **게임 오버**.
5. **레벨 1**에는 손가락 안내가 있는 **튜토리얼**(가로 → 세로 한 줄)이 한 번 나옵니다.

## 주요 기능

### 퍼즐 & 진행
- **레벨 2000개** — 동물·사람·음식·물건·식물 등 알아볼 수 있는 픽셀 그림. 5의 배수(15 이상)는 컬러풀한 **보스 보석** 레벨
- **벨트 발사 메커니즘** — 완성한 줄의 색 블록이 상단을 회전하며 같은 색 픽셀을 제거
- **대기열 시스템** — 못 쏜 블록은 대기열로, 가득 차면 게임 오버
- **콤보 시스템** — 연속 줄 클리어 시 콤보(3칸 유예 게이지)
- **스토리 모드** — 레벨마다 진행되는 유리·준의 연애 대화 + **음성(TTS)**·캐릭터 초상화

### 코인 · 아이템
- **클리어 룰렛** — 클리어 시 코인 룰렛, **광고 보고 2배 받기**(가상 광고)
- **아이템 4종** — 대기열 +1 · 한 칸 블록으로 변환 · 새 블록 · 2×2 폭탄
- 위험할 때(대기열 절반↑ / 보드 빈자리 부족) 아이템이 **반짝이며 사용 안내**

### 픽셀 컬러링 🎨
- 플레이로 **획득한 색만큼** 칠하는 컬러링 모드 (도안 **37종**, 4열 갤러리)
- **한 손가락 드래그로 색칠**(숫자 칸 위) · 두 손가락으로 확대·축소·이동 · 연필 커서
- 고정 **지우개**, 도안별 **초기화 / 되돌리기 / 완성 미리보기**

### 방 꾸미기 🏠
- 레벨 10개 클리어마다 한 방씩, **총 200개 방**(가구 2000개)
- 방마다 **진행 게이지 + 완성 시 500코인**
- 전체 진행 게이지에 **마일스톤 말풍선**(40·80·120·160·200방 → 🪙1,000 / 2,000 / 3,000 / 5,000 / 10,000)
- 방을 모두 꾸미면 보상 화면

### 홈 & 기타
- **PLAY**(이어하기) · **FREE 맵 골라하기**(드래그 스크롤·탭 선택) · **AUTO 자동 테스트**
- **사운드 / 진동** 켜고 끄기, **초기화 버튼**(방·코인·컬러링·진행도 전체 리셋)
- **AUTO 자동 테스트** — AI가 자동 플레이, 실행 중 배속(1·2·4·16x) 변경, 화면 ON/OFF, 레벨별 난이도 산출 + 엑셀(xls) 로그
- **모바일 지원** — 9:16 세로 화면, 터치 조작

## 레벨 데이터 (JSON)

모든 레벨은 `levels/` 폴더에 **JSON 파일 1개씩**(`level_001.json` … `level_2000.json`) 들어 있어
그대로 보거나 수정할 수 있습니다. 자세한 형식은 [`levels/README.md`](levels/README.md) 참고.

- 콘솔에서 `G.loadLevelJSON(<파일내용>)` 호출, 또는
- 주소 뒤에 `?play=levels/level_042.json` 를 붙여 접속하면 그 레벨이 바로 시작됩니다.

## 실행 방법

### 1) 파일 바로 열기
`pixel_block_puzzle.html` 을 브라우저에서 열기만 하면 됩니다. 설치·빌드 없음.

### 2) 로컬 서버 (PowerShell)
```powershell
./serve.ps1            # 기본 포트 8849
./serve.ps1 -Port 3000 # 포트 지정
```
실행 후 `http://localhost:8849/` 접속.
