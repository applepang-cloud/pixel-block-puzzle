# Pixel Block Puzzle 🧩

**마음속 세포들의 로맨스**가 담긴 블록 퍼즐 게임. 외부 라이브러리 없이 순수 HTML/CSS/JavaScript 단일 파일(`pixel_block_puzzle.html`)로 동작합니다.

블록을 끌어다 줄을 완성하고, 레벨을 깨며 펼쳐지는 세포들의 풋풋한 연애 이야기를 즐겨보세요. ♥

## 게임 소개

레벨을 클리어할 때마다 **비주얼 노벨** 형식의 대화가 한 컷씩 진행됩니다. 부끄럼세포·용기세포 등 마음속 세포들이 설렘 가득한 로맨스를 이끌어 갑니다. (유미의 세포들 스타일)

## 주요 기능

- **레벨 1~10** — 단계별로 난이도가 올라가는 도전
- **드래그 & 드롭 퍼즐** — 하단 조각을 끌어다 줄(라인)을 완성
- **벨트 발사 메커니즘** — 완성된 줄 블록이 상단 벨트를 **3바퀴** 돌며 같은 색 픽셀을 발사
- **대기열 시스템** — 픽셀을 맞추지 못하면 대기열로 이동, 대기열이 가득 차면 게임 오버
- **스토리 모드** — 레벨마다 한 컷씩 진행되는 세포들의 로맨스 대화
- **내 방 꾸미기** — 레벨을 1개 클리어할 때마다 가구 1개 획득, 모은 가구로 방 꾸미기 (침대·소파·액자·무드등·리본 장식 등)
- **효과음 & 음소거** — 우측 상단 버튼으로 켜고 끄기
- **모바일 지원** — 터치 드래그로 조작, 반응형 화면

## 플레이 방법

1. 화면 **아래 블록을 끌어다** 가로 줄을 완성하세요.
2. 완성된 줄 블록이 **상단 벨트를 돌며 같은 색 픽셀을 발사**합니다.
3. 발사된 픽셀을 잘 맞추세요 — 못 맞추면 **대기열**로 들어갑니다.
4. **대기열이 가득 차거나** 블록을 놓을 곳이 없으면 게임 오버!
5. 레벨을 클리어해 스토리를 진행하고, 가구를 모아 나만의 방을 꾸며보세요.

## 실행 방법

`pixel_block_puzzle.html` 파일을 브라우저에서 열기만 하면 됩니다. 설치나 빌드 과정이 전혀 없습니다.

## 기술

- 순수 HTML · CSS · JavaScript (단일 파일, 의존성 없음)
- `<canvas>` 기반 렌더링
- WebAudio 효과음
