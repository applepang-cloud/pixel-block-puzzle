# 레벨 JSON 폴더 📂

각 레벨이 **JSON 파일 1개**로 들어 있습니다. `level_001.json` ~ `level_900.json`.
이 파일을 주시면 그대로 **플레이**하거나 **수정**할 수 있어요.

- 1~112: 초반 레벨
- 113~512: 알아볼 수 있는 그림(동물·사람·음식·물건·식물). 5의 배수는 컬러풀 보스
- 513~900: 기존 113~500 레벨(이동)

## 파일 형식

```jsonc
{
  "level": 42,          // 레벨 번호
  "hard": false,        // 하드(보스) 레벨 여부 (5의 배수, 15 이상)
  "rows": 11,           // 그림 세로 칸 수
  "cols": 13,           // 그림 가로 칸 수
  "palette": {          // 이 레벨에서 쓰는 색(참고용)
    "R": "#ff4757",
    "Y": "#ffd700"
  },
  "grid": [             // 그림. 한 줄 = 한 행. 글자 1개 = 한 칸.
    ".....RR......",    //   '.' = 빈 칸
    "....RYYR....",     //   색 글자 = 그 색 픽셀
    "..RRYYYYRR.."
  ],
  "story": "💭 ...",    // 상단 스토리 한 줄 (선택)
  "dialogue": {         // 대화 (선택, 없으면 null)
    "intro": [ {"who":"H","face":"😊","text":"...","name":"유리"} ],
    "outro": [ {"who":"M","face":"🥰","text":"...","name":"준"} ]
  }
}
```

### 색 글자 (color keys)

| 글자 | 색 |  | 글자 | 색 |
|---|---|---|---|---|
| `R` | 빨강 | | `O` | 주황 |
| `G` | 초록 | | `C` | 하늘(청록) |
| `B` | 파랑 | | `W` | 흰색 |
| `Y` | 노랑 | | `K` | 어두운색(검정) |
| `P` | 보라 | | `N` | 분홍 |

(`.` 은 빈 칸) — 전체 색 코드는 `_colors.json` 참고.

## 수정하는 법
- `grid` 의 글자를 바꿔 그림을 새로 그리면 됩니다. `rows`/`cols` 는 그림 크기에 맞추세요.
- `grid` 의 각 줄 길이는 `cols` 와 같아야 합니다(짧으면 빈 칸으로 채워짐).
- 모든 픽셀은 가장자리에서 같은 색을 쏴서 지울 수 있어야 클리어됩니다(보통 꽉 찬 그림은 OK).

## 플레이하는 법
- 게임을 열고 콘솔에서 `G.loadLevelJSON(<파일내용>)` 호출, 또는
- URL 뒤에 `?play=levels/level_042.json` 를 붙여 접속하면 그 레벨이 바로 시작됩니다.

> 이 파일들은 게임 본체(`pixel_block_puzzle.html`)에서 자동 추출되었습니다.
> 규칙·색은 게임과 동일합니다.
